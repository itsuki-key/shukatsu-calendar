# 28卒サマーインターン カレンダー

商社・デベロッパー・コンサル・金融・広告・ベンチャー/ITの主要企業の2026年サマーインターン情報を、月別カレンダーと企業別フローで一覧できるWebページです。

## 主な機能

- 業界別カラーで月別タイムラインを表示
- 企業別の選考フロー（エントリー→選考→インターン→本選考優遇）
- 各イベントから公式マイページ/採用ページに直接ジャンプ
- **「📅 追加」ボタンで、興味のあるイベントだけ自分のGoogleカレンダーに追加** (Google calendarには自分以外見えないのでプライバシー安全)

## ファイル構成

```
intern-calendar/
├── index.html   # 見た目とレイアウト（基本いじらない）
├── data.js      # ★ 企業・日程・フローのデータ（更新時はここだけ差し替え）
├── app.js       # 表示ロジック（基本いじらない）
└── README.md    # この説明
```

## 公開方法（GitHub Pages）

1. GitHubで新しいリポジトリを作成（例: `intern-calendar`、Publicを選択）
2. 4つのファイルをまるごとアップロード
3. リポジトリの `Settings` → `Pages` を開く
4. `Source` を `Deploy from a branch`、`Branch` を `main` `/ (root)` に設定して `Save`
5. 数十秒〜数分後に `https://<your-username>.github.io/intern-calendar/` で公開される
6. このURLを友達に共有・自分でブックマーク

## Googleカレンダー追加について

月別カレンダータブで各イベント右端の「📅 追加」をクリックすると、Googleカレンダーの「予定を作成」画面が新タブで開きます。

- タイトル、日時、説明、場所(公式URL)が事前入力されているので、保存ボタンを押すだけ
- **自分のGoogleアカウントだけに追加される** (他の人には見えない)
- 通知時間はGoogleカレンダー側で設定 (おすすめ: 終日イベントは前日18時に通知)
- 「通年」など日付が未確定のイベントは「📅 —」と表示され、追加できません
- 「6月中旬」のような表記は中旬=15日、上旬=5日、下旬=25日として登録されます

## 更新の流れ

新しい企業を足したい、締切が判明したなど、更新したいときは:

1. Claudeに「最新情報に更新して、◯◯と△△も足して」と依頼
2. Claudeが `data.js` の更新版を出してくれる
3. GitHubでファイル `data.js` を開いて鉛筆アイコンで編集モード、内容を貼り替えて `Commit changes`
4. URLは変わらず、中身だけ最新版に。友達のブックマークもそのまま使える

## ローカルで動作確認したい場合

ファイルをダウンロードして `index.html` をブラウザで直接開けば見られます。
